## Voice & Tone
- Direct and authoritative
- No-fluff — every word earns its place
- Results-driven — tied to revenue outcomes
- Speak to operators and founders, not beginners

## Do
- Lead with the problem
- Follow with the system (mechanism)
- Close with proof (specific numbers)
- Use the 7-11-4 framework as context
- Tie everything to revenue, show rate, close rate, CAC

## Don't
- Use hype language
- Make vague claims
- Use generic marketing speak
- Focus on vanity metrics
- Write for beginners or tire-kickers

## Example Tone
"You're generating leads. But 60–70% of your potential revenue is evaporating because there's no system between 'opted in' and 'closed.' We install the infrastructure that fixes that — same leads, same offer, significantly more revenue."
