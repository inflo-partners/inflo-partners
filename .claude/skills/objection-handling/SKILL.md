# Objection Handling — Reference

## Universal Objections (All Verticals)

| Objection | Reframe |
|---|---|
| "It's too expensive" | Frame cost against cost of inaction. "You're already losing $X/month by not fixing this." |
| "I need to think about it" | "What specifically do you need to think about? Let's address it now." Provide more proof. |
| "I already have someone for this" | "Most clients come to us with existing providers. The gap is usually in coordination, not talent." |
| "Is this really free?" | "Yes. You'll leave with clarity on your situation either way. We only work with people we can deliver results for." |
| "I don't have time" | "This is designed to save you time. The 30-minute call identifies what's costing you the most right now." |
| "How do I know this works?" | Stack proof — case studies, specific numbers, retention rate, industry credibility. |

## Financial Advisory / Wealth Management

| Objection | Reframe |
|---|---|
| "I already have a wealth manager" | "We coordinate across all advisors — tax, estate, investments — where the real gaps are." |
| "Will you move my assets?" | "Not unless it makes sense. We focus on strategy, not asset gathering." |
| "What's the minimum?" | State it clearly. Qualifying is a trust signal. |

## Coaching / Fitness / Info Products

| Objection | Reframe |
|---|---|
| "I've tried programs before" | "Those programs gave you information. We install a system with accountability." |
| "Can't I just do this myself?" | "You could. But how long have you been trying? The system compresses the timeline." |
| "What if it doesn't work for me?" | Risk reversal — guarantee, results timeline, proof from similar clients. |

## SaaS / B2B

| Objection | Reframe |
|---|---|
| "We're already using [competitor]" | "We integrate with or replace — let's look at what's actually driving results vs. what's just running." |
| "Need to get buy-in from my team" | "Happy to do a quick walkthrough with your team. What's the best way to set that up?" |
| "What's the ROI?" | Show specific metrics from similar companies. Frame as revenue impact, not cost. |

## When to Use Objection Handling
- **Landing pages**: FAQ section (preemptive)
- **Email sequences**: Dedicated objection-handling email (Email 3)
- **VSLs**: "The Mistake" section + proof stacking
- **SMS**: Post-call follow-up when lead goes cold
- **Ads**: Retargeting copy addressing specific hesitations
